# Crystal's Cakes - Full-Stack E-Commerce Website

Welcome to Crystal's Cakes! This is a full-stack e-commerce website built using the following technologies:

- **Frontend**:
  - HTML
  - CSS (Styling)
  - Javascript

- **Backend**:
  - Django (Python web framework)
  - PostgreSQL (Database)
  - RESTful APIs
